# Crypto Vault

## About

Crypto Vault allows to store encrypted data on-chain

Contract address at mumbai testnet: `0x276c89d95300b11b8ACEAE24a2dbc4bc3BaB69F5`

## Usage

### Build

```bash
forge build
```

### Test

```bash
forge test
```

### Deploy

```bash
make deploy ARGS='--network mumbai'
```
